演奏コマンドパッチ　eraTW4.37proto用　β版

TWでモスボール化されていた演奏コマンドを使用できるようにしたものです。
調整方針としては辛めの方針です。

凍結箇所からの変更は
・ITEMの楽器類のコメントアウト解除
・COMF416の調整
	‥キーボードが屋「外」でのみ使用可能の理由が不明だったので屋内外で使用可能に
	‥ピアノが何故かチェンバロになっている箇所を修正
	‥ピアノは博麗神社居間、紅魔館の広場と廃洋館、地霊殿の広場、寺子屋に設置しているという形に
		…サブマップ時の紅魔館の広場と廃洋館が何故かMAIN_MAP != 1 && CFLAG:MASTER:現在位置 == 廃洋館 || MAIN_MAP != 1 && CFLAG:MASTER:現在位置 == 広間 ||を読み込まないため未対応です。
		…サブマップ時の博麗神社にも未対応です。
		…そもそも博麗神社にピアノがあるイメージが無かったので凍結時から触っていません…申し訳ありません
	‥ITEMとRESULT対応の所が1つずつずれていたので修正
	‥演奏成功時に演奏経験ボーナス+5
	‥楽器持ちキャラ、プリリバ三姉妹と九十九姉妹と雷鼓がTARGETで彼女らの音楽技能を上回っている場合にTARGETとあなたに演奏経験ボーナス
	‥歌唱キャラ、ミスティアと響子、エレン（口上で歌っていたため）がTARGETで彼女らの音楽技能を上回っている場合にあなたに演奏経験ボーナスとTARGETに歌唱経験ボーナス
	‥TARGETの音楽技能があなた「以上」であるかTARGETが音楽知識所有
	‥音楽知識所有の場合演奏経験ボーナス
	‥音感所持の場合演奏経験ボーナス
	‥音楽技能が1以上ある場合それぞれ1ごとに演奏経験ボーナス+1
	‥添い寝中、睡眠中、仕事中、時間停止中は演奏できません
	‥人前60分間隔で演奏できます。連続して演奏するとしんどい。（演奏そのものに30分かかるので実質30分間隔）
		…一人の場合練習のため連続して演奏できます。
・TRACHECK＿ABLUP.ERBの音楽技能関連（ヴァイオリンと音楽知識、音感）のSKIPとコメントアウトを解除
	‥音楽知識取得条件の演奏経験5000を500に変更
	‥音感取得条件の演奏経験50000を1000に変更
・EVENT_MESSAGE_COM400内の口上を一部変更
	‥楽器持ちキャラ、プリリバ三姉妹と九十九姉妹と雷鼓がTARGETで彼女らの音楽技能を上回っている場合
	‥歌唱キャラ、ミスティアと響子、エレン（口上で歌っていたため）がTARGETで彼女らの音楽技能を上回っている場合
	‥TARGETの音楽技能があなた「以上」であるかTARGETが音楽知識所有
	‥その他
	で地の文が少し変わります。
