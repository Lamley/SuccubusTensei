TW 4.45修正3用 スレ>>614 お米表示＆通販購入改善パッチ(案)
以下をよく読んで、わかる人だけ使ってみてください。

変更内容:
・雑穀・古米・白米などが持ち物としてきちんと表示されるようにしました。
	  PRINTFORMLC と PRINTCPERLINE() を使うようにしたので、使用上好みがわかれると思います。
	  項目の幅と一行当たりの項目数は、メニューの「ヘルプ」→「設定」の「表示」タブにある
	「PRINTCを並べる数」と「PRINTCの文字数」を調整してみてください。
	  並べる数は 3～5、文字数は 24～28前後が良いかと。

・メニューの通信販売で雑穀が購入ができるようにしました。

・お米と苗(通販とお店共通)、お酒(通販のみ)での在庫数量に上限を設定しました
	  毎日一定量入荷されます。
	お願い:
	  毎日の入荷量は、SHOP_ITEM.ERB内の@EVENTTURNENDに設定しています。
	  量は暫定なので、適度な値に設定してください。

・変更に伴い、河童の道具屋と花屋での一覧表示も変わりました
